CREATE DATABASE registros_db;

USE registros_db;

CREATE TABLE registros (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(255) NOT NULL,
    edad INT NOT NULL
);


Para crear el entorno en Python:

python -m venv venv

Para crear el archivo con las extenciones instaladas:

pip freeze > requirements.txt

Para instalar las extenciones hacer lo siguiente:

pip install -r requirements.txt
